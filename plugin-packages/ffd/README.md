# Galacean Effects Free-form deformation Plugin

## Usage

### Simple Import

``` ts
import { Player } from '@galacean/effects';
import '@galacean/effects-plugin-ffd';
```

## Development

### Getting Started

``` bash
# demo
pnpm --filter @galacean/effects-plugin-ffd dev
```

> [Open in browser](http://localhost:8081/demo/)

# Galacean Effects KTX2 Plugin

## Usage

### Simple Import

``` ts
import { Player } from '@galacean/effects';
import '@galacean/effects-plugin-ktx2';
```

## Development

### Getting Started

``` bash
# demo
pnpm --filter @galacean/effects-plugin-ktx2 dev
```

> [Open in browser](http://localhost:8081/demo/)
